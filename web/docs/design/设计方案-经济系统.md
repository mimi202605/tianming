### 四子系统职责

| 子系统 | 管辖 | 与其他子系统的关系 |
|--------|------|-------------------|
| **财政系统** | 国家岁入岁出（帑廪·内帑·三列账）、税种、诏令扣账 | 货币系统决定账本"购买力"；央地决定账本数字的分层归属；角色决定税收效率/抄家所得 |
| **货币系统** | 铸币、成色、纸币、准备金、通胀、钱荒、银流 | 改变财政库存的实际购买力；影响角色家产估值；影响央地分账 |
| **央地财政** | 中央/各级地方/封建领地的分账、上供/留使/留州、央地博弈 | 财政系统按此分层；地方截流体现在 P.2 百姓负担模型 |
| **角色经济** | 每个 NPC 的俸禄、家产、贪腐收入、家族经济 | 影响税收效率（N.4）；抄家时实际查获由角色真实家产决定 |

### 决策框架（待你确认后进入细节）

| # | 决策 | 备选 |
|---|------|------|
| Ⅰ | 币种范围 | 金/银/铜/铁/贝/纸 六类全建模 vs 按剧本朝代筛选 vs 仅核心三种（银铜纸）|
| Ⅱ | 本位制 | 铜本位 / 银本位 / 双本位 / 纸币本位（按剧本朝代配置）|
| Ⅲ | 铸币机制深度 | 只算数量 / 加成色降级 / 加私铸劣币 / 全模型 |
| Ⅳ | 纸币机制 | 不做 / 准备金+信用度简单模型 / 完整纸币生命周期模型 |
| Ⅴ | 物价动态 | 静态比价 / 按年景动态 / 完整市场博弈 |
| Ⅵ | 海外银流 | 不做 / 按朝代事件触发 / 持续动态建模（宋海贸、明清白银流入流出）|
| Ⅶ | 货币改革作为玩家动作 | 不做 / 允许改铸/发钞/废钱 / 全部改革可做且有历史预设 |
| Ⅷ | 地域币值差异 | 不做 / 南北两分 / 各路/省独立币值 |
