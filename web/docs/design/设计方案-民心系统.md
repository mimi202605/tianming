# 设计方案 · 民心系统（天视自我民视）

> 第四官方变量。7 大官方变量中最接近"天命根基"的核心。  
> **定位**：民心不是一个分数，而是**朝廷视野与天下实情的裂痕**——与腐败系统同构的信息失真哲学。  
> **哲学**：得民心者得天下。朝廷看到的民心永远是被地方粉饰、被胥吏过滤、被祥瑞掩盖的版本。皇帝的第一场战争，是与"不知道"搏斗。

---

## 零、核心哲学

### 0.1 三条铁律

1. **民心不是单一数值，是两张图**  
   一张是**真实民心图**（trueMap）——天下实情；另一张是**朝廷视野图**（perceivedMap）——皇帝所见。两图之间的鸿沟决定了皇帝决策是否脱离地面。

2. **民心是物理现象，不是惩罚**  
   民心低不直接扣分，而是自然转化为：实征率下降、征兵困难、逃亡率升、谶纬流传、地方叛附倾向。民心高时，政令畅通、改革可行。

3. **民变是渐变过程，不是突发事件**  
   民心从 60 跌到 10 不是一日之功——流言→聚啸→暴动→起义→改朝换代，五级递进。早期干预便宜，晚期代价王朝级。

### 0.2 与其他系统的关系

- **腐败**：腐败是民心崩溃的最大隐性来源（实征漏损 + 司法不公 + 地方官勒索）。但民心崩溃的**感知**又被腐败进一步扭曲——地方官不会如实奏报民怨。
- **风闻录事**：民变征兆（流言、盗匪、聚啸）全部进入风闻四类（告状/风议/密札/耳报）。监察越弱，风闻越偏颇。
- **户口**：徭役与隐户是民心传导的重要中介。隐户暴增意味着民心"**用脚投票**"。
- **威望**：威望是皇帝的个人声誉，民心是百姓对王朝的态度。二者强相关但非等同。

### 1.3 真实值 vs 感知值（C2）

```javascript
function updatePerceivedMinxin(dept) {
  var localCorr = GM.corruption.subDepts.provincial.true;
  var sup = GM.corruption.supervision.level;
  
  // 粉饰偏差：地方粉饰——朝廷看到的比实际高
  var biasMax = (100 - sup) * 0.3 + localCorr * 0.3;
  var bias = Math.random() * biasMax;  // 始终正向（向好粉饰）
  
  // 极端情况：某些地区可能反向（煽动性奏报——为了自己功劳要赈灾)
  if (localOfficial.incentive === 'seekReliefFund') bias = -bias * 0.5;
  
  return clamp(trueV + bias, 0, 100);
}
```

**关键**：监察弱或腐败高时，perceived 可以比 true 高 **20-30 点**——这正是历代末世的真实情形："朝廷犹谓天下承平，而民已大乱"。

### 1.4 分区 × 分阶层矩阵（小型交叉表）

内部维护 `matrix[region][class] = { true, perceived }`，但 UI 层通常汇总为 `byRegion` 或 `byClass` 两个视图之一。

矩阵汇总到全局：

```javascript
function aggregateMinxin() {
  var sum = 0, totalWeight = 0;
  for (var r in byRegion) {
    var regionPop = getRegionPopulation(r);
    var regionIdx = byRegion[r].true;
    sum += regionIdx * regionPop;
    totalWeight += regionPop;
  }
  GM.minxin.trueIndex = sum / totalWeight;
  GM.minxin.phase = matchPhase(GM.minxin.trueIndex);
}
```

## 二、来源机制（14 类）

### 2.1 赋税实征（taxation）

**民心受"实缴"而非"名义税"影响**——与财政系统的三数对照呼应。

```javascript
function calcTaxImpactOnMinxin() {
  var actualBurden = GM.fiscal.peasantActualPayment;   // 民间实缴
  var ability = GM.minxin.payAbility;                  // 民间支付能力
  var ratio = actualBurden / ability;
  
  // ratio=1.0 → 0 影响
  // ratio=1.3 → -5
  // ratio=2.0 → -20
  // ratio=3.0 → -40
  if (ratio <= 1.0) return +3 * (1 - ratio);  // 轻税有正面影响
  return -Math.pow(ratio - 1, 1.5) * 15;
}
```

### 2.2 徭役（corvee）

```javascript
function calcCorveeImpact() {
  var corveeDays = GM.corvee.annualDaysPerCapita;
  // 每年 15 日内正常；超过线性恶化
  if (corveeDays <= 15) return 0;
  return -(corveeDays - 15) * 0.4;
}
```

**历史参照**：秦汉岁 30 日为底，隋炀帝时期严重超标。

### 2.3 灾荒赈济（disasterRelief）

```javascript
function calcReliefImpact() {
  var active = GM.activeDisasters;
  var total = 0;
  for (var d of active) {
    var reliefCoverage = d.reliefFund / d.estimatedNeed;
    var timelyFactor = d.turnsSinceStart <= 2 ? 1.0 :
                       d.turnsSinceStart <= 5 ? 0.6 : 0.2;
    
    // 充分及时：+10；完全不赈：-25
    var impact = reliefCoverage >= 1 && timelyFactor >= 1 ? 10 :
                 reliefCoverage >= 0.5 ? reliefCoverage * timelyFactor * 8 :
                 -25 * (1 - reliefCoverage) * (1 - timelyFactor);
    total += impact;
  }
  return total;
}
```

**历史反面**：明末崇祯时期，陕西大旱官方赈济几近于无，酿成李自成起义。

```javascript
function calcPriceImpact() {
```

### 2.8 阶层流动（socialMobility）

```javascript
function calcSocialMobilityImpact() {
  var examOpen = GM.officialSystem.imperialExamActive ? 1 : 0.3;
  var yinBuRatio = GM.officialSystem.yinBuRatio;  // 荫补越多 → 流动性越低
```

### 2.9 文化宗教政策（culturalPolicy）

```javascript
function calcCulturalImpact() {
  var persecution = GM.ongoingReligiousPersecutions;   // 灭佛、禁道、禁基督
  var ethnicTension = GM.ethnicPolicyTension;
  
  var impact = 0;
  for (var p of persecution) {
    impact -= p.severity * 2;   // 灭佛/禁道会让信众民心大减
    // 但儒家士人可能欢迎
    if (p.target === 'buddhism' && GM.dominantIdeology === 'confucian') {
      impact += p.severity * 0.5;
    }
  }
  impact -= ethnicTension * 0.3;
  return impact;
}
```

**历史参照**：三武一宗灭佛、清代禁天主教、元代崇番抑汉

### 2.10 天象异常（heavenSign）

```javascript
function applyHeavenSign(sign) {
  // 日食：-8 ~ -15（视位置/年份/对皇帝名字的联想）
  // 彗星：-10 ~ -20
  // 地震：-5 ~ -30（视震级与都城距离）
  // 蝗灾：-15 ~ -30（尤其粮产区）
  // 血雨/陨石/怪风：-5 ~ -20
  
  var impact = sign.baseImpact;
  if (GM.eraState.dynastyPhase === 'decline' || 'collapse') impact *= 1.5;
  if (GM.eraState.dynastyPhase === 'peak') impact *= 0.5;
  
  // 皇帝是否有德政回应（罪己诏/免赋/赈灾）
  if (sign.responded) impact *= 0.4;
  
  return impact;
}
```

### 2.11 祥瑞（auspicious）

```javascript
function applyAuspicious(sign) {
  // 瑞雪：+3
  // 嘉禾（一茎多穗）：+4
  // 白雉：+6
  // 麒麟出现：+15（重大）
  // 甘露降：+5
  // 凤凰至：+20（改朝换代级）
  
  var impact = sign.baseImpact;
  
  // 真 vs 伪：若皇帝近期"营造祥瑞"（F8），短期生效但长期透支
  if (sign.fabricated) {
    impact *= 0.6;
    // 若被识破
    if (sign.exposedRisk > Math.random()) {
      return -impact * 2;  // 反噬
    }
  }
  
  return impact;
}
```

### 2.13 战事胜负（warResult)

```javascript
function calcWarImpact() {
  var recent = GM.recentWarResults;
  var impact = 0;
  for (var r of recent) {
    impact += r.victory ? 5 : -8;
    if (r.victory && r.againstBarbarian) impact += 5;  // 驱逐外族鼓舞民心
    if (r.defeat && r.inTerritory) impact -= 10;       // 本土战败尤其惨
  }
  return impact;
}
```

```javascript
function calcImperialVirtueImpact() {

  for (var a of acts) {
    if (a.type === 'mourn-disaster')        impact += 2;
    if (a.type === 'confession-edict')      impact += 4;  // 罪己诏
```

```javascript
function calcReformTolerance() {
```

## 四、提升手段（14 类）

### 4.1 减税/免税


### 6.4 升级公式

```javascript
function checkUprisingEscalation(uprising) {
  var r = uprising.region;
  var localMinxin = GM.minxin.byRegion[r].true;
  
  // 基础推进：民心越低，升级越快
  var escalationRate = (40 - localMinxin) / 100 * 0.2;
  
  // 领袖魅力加速
  escalationRate += uprising.leader.charisma / 500;
  
  // 朝廷应对削弱
  if (uprising.hasImperialResponse) escalationRate *= 0.5;
  if (uprising.militarySuppression > followers / 2) escalationRate *= 0.2;
  
  // 同期多地起义互相激励
  var otherUprisings = GM.minxin.uprisings.filter(u => 
    u.id !== uprising.id && u.level >= uprising.level);
  escalationRate += otherUprisings.length * 0.05;
  
  uprising.momentum += escalationRate * 10;
  
  if (uprising.momentum >= 100) {
    uprising.level++;
    uprising.momentum = 50;
    triggerEscalationEvent(uprising);
  }
}
```

### 6.5 干预选项（按级别）

#### L1 流言期
- **派员查访**（轻）：成本低，20% 化解
- **张贴安民告示**（轻）：民心 +1，30% 化解
- **遣特务监察**：若有厂卫，60% 化解但粗暴
- **听之任之**：50% 自行消散 / 50% 升级

#### L2 聚啸期
- **地方官调解**（怀柔）：60% 化解但赈济代价
- **派员弹压**：80% 化解但激化风险
- **允许自首免罪**：50% 化解
- **无动于衷**：80% 升级

#### L3 暴动期
- **调州兵**：成功率 ≈ 40-70%（看质量）
- **派钦差**：外交+军事组合
- **招安**：给首领官职（可能背叛）
- **重兵围剿**：成本高但彻底
- **转嫁矛盾**：如让豪强自备私军

#### L4 起义期
- **调中央军**（京畿边军）：王朝级动员
- **封爵招安**（给巨大好处）：可能反叛
- **分而治之**：离间首领
- **决战**：胜败决定王朝命运
- **迁都避锋**：暂避但失人心

#### L5 改朝换代
- 这是**王朝危机**级游戏事件，玩家需在多线作战
- 失败即**游戏事件**（王朝终结或陷入分裂）

## 八、与其他变量联动

### 8.1 腐败 → 民心（**强传导**）
- 实征漏损（Δ名义-官收）加倍压在民间 → 浮收
- 司法腐败 → 冤狱 → 民心
- 地方官腐败 → 勒索 → 民心

### 8.2 帑廪 → 民心（强）
- 赋税实征决定民间承担
- 专项支出（工程/军费）的徭役负担

### 8.3 威望 → 民心（中双向）
- 军事胜利 → 民心
- 民心高 → 皇帝威望
- 威望低 → 民间疑君失德 → 民心

### 8.4 皇权 → 民心（弱但关键时刻强）
- 皇权过强（酷烈）→ 民心 / 士人心离
- 皇权过弱（地方豪强盘剥）→ 民心 / 农民心离
- 皇权稳定 → 民心常态化

### 8.5 户口 → 民心（中）
- 徭役重 → 民心
- 隐户多 → 民心用脚投票的表现

### 8.6 制度 → 民心（情境）
- 改革恰当 → 民心
- 改革激进 → 民心（视容忍度）
- 开倒车 → 民心（开明阶层离心）

### 8.7 军事 → 民心（双向）
- 胜仗：民心 +
- 本土作战：民心 -（战事扰民）
- 征兵沉重：民心 -
- 军纪涣散（军队腐败）：民心 --

### 8.8 风闻录事 → 民心感知
- 告状 → 玩家看到民怨征兆
- 风议 → 士人心态指标
- 密札 → 内情透露
- 耳报 → 可能诬告

### 8.9 腐败 ↔ 民心双向
- 腐败推高民怨
- 民怨倒逼玩家肃贪
- 肃贪有副作用（党争）
- 党争反过来让地方官更不作为 → 民心继续跌

## 十五、设计特色

1. **天下视野 vs 朝廷视野的二元对照**——民心从来不是一个数，而是两张图：真实的、粉饰的。肃贪和监察不只是为了抓贪官，更是为了**知道**。

2. **五级升级链而非突发事件**——民心从 60 到 10 不是一次下跌，而是流言→聚啸→暴动→起义→改朝的完整递进。玩家早期干预便宜，晚期代价王朝级。

3. **异象系统的三元结构**——天象（惩罚）+ 祥瑞（赞颂）+ 谶纬（预言）。三者构成"天人感应"的完整叙事机制，让中国历史独特的政治哲学在游戏中成立。

4. **分区 × 分阶层的细粒度**——同一时刻江南可能安居、陕西可能揭竿；士人可能颂圣、流民可能将反。这是唯一能模拟"同治中兴"（地方中兴）与"末世共存"的设计。

5. **14 源中包含"文化宗教"与"皇帝德行"**——非物质因素参与民心计算。灭佛、禁教会失教徒心；罪己诏/亲临灾区会感召民心。这些是史书上反复出现的真实机制。

6. **所有提升手段皆有代价**——减赋→帑廪；大赦→治安；罪己→皇权；祥瑞→长期透支；打击豪强→党争；开科→贤能长期回报。没有免费的民心。

7. **粉饰文本 AI 生成**——奏疏系统让玩家看到的是地方官报喜不报忧的版本。监察强时才能看到真相。这完美呼应历史上"朝廷犹谓天下承平，而民已大乱"的情境。

8. **与风闻录事深度绑定**——4 类风闻（告状/风议/密札/耳报）就是民心信号的具体形态。credibility 决定玩家能否基于它作决定。

9. **历史预设对照 12 朝**——从秦到清每个朝代每个阶段的民心范围有历史依据，让"同一时代背景的不同选择"可对比。

10. **与腐败系统完美对偶**——腐败关注吏治（皇权 vs 官僚），民心关注民意（朝廷 vs 百姓）。二者均有"真实 vs 感知"机制，构成对称。
