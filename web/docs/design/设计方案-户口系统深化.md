### A2 承载力综合公式

```js
region.carryingCapacity.max = Math.min(
  region.farmland × yieldPerUnit × 0.8 / capitaNeed,    // 耕地决定的上限
  region.waterCapacity × 0.7 / capitaWaterNeed,          // 水源上限
  region.climateFactor × baseline                         // 气候上限
);

// 超载触发
if (currentPop > capacity.max × 1.1) → 饥荒概率 +, 迁徙出 +
if (currentPop < capacity.max × 0.3) → 迁徙吸引 +, 耕地荒芜
```

### B2 常役 vs 大工程 vs 战时特征

**常役**（每年常态）：
- 年役日 = 丁年役日标准（30 日左右）
- 按丁摊派
- 折银可行

**大工程**（诏令触发，限期）：
- 集中征调数万~数十万丁
- 限期几年内完工
- 完工/烂尾/延期 三种结果

**战时特征**（战争期间临时增加）：
- 军需运输
- 战事后勤
- 可强征 2-3 倍正常工日

### C5 军队调动（完整）

```js
function _calculateMarch(army, fromRegion, toRegion, options) {
  var distance = _regionDistance(fromRegion, toRegion);
  var terrain = _terrainBetween(fromRegion, toRegion);
  
  return {
    durationTurns: distance / army.marchSpeed / terrain.speedMod,
    grainConsumption: army.size × distance × dailyGrainPerSoldier,
    moraleDecay: distance × 0.001 × (terrain.harsh ? 2 : 1),
    terrainRiskEvents: [                       // 沿途风险
      { at:turn1, type:'水难', probability:0.05 },
      { at:turn3, type:'贼扰', probability:0.1 }
    ],
    fatigueOnArrival: 0.1 + distance × 0.001,
    escortRequired: army.size × 0.1             // 护送后勤
  };
}
```
