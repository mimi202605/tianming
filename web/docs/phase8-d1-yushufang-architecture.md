#### ③ 御书案 (主交互·~50%)

```
案上 7 类物·each = 1 操作入口·

[砚台]·圆形·黑润 (端砚 / 歙砚)
  click → 画面 zoom 入砚池
       → 浮 5 笔类印 (政 / 军 / 外 / 经 / 他)·篆体小印
       → 选 1 印 → 砚台前展卷 (御笔题"奉天承运皇帝诏曰") → 写诏 textarea
       → 完成 → click 朱印 (旁) → 盖印动画 → 卷自动卷归档
  对应·gt-edict (5 类诏令编辑)
  状态·砚池有水 = 可写·无水 = 缺 (能量 0)

[笔架]·两支毛笔搁
  hover·浮·"御 笔"
  click·选笔类 (alt 砚台进入)
  对应·gt-edict 笔类切

[镇纸]·一对·青铜兽形 (狻猊 / 麒麟)
  hover·浮·"奏 疏 待 览·N 折"·N 跟 GM.memorials.length
  click·镇纸抬起 → 案右侧 折子叠 现 (~5-15 折堆)
       → click 1 折 → 展开 → 朱批 / 放回 / 留中
  对应·gt-memorial

[朱印]·朱泥湿润印 + 印章
  hover·浮·"用 玺"
  click·当前若有未完诏书 → 盖印动画 → 完成
       若无·浮·"无诏可印"
  对应·诏令颁行

[印泥盒]·圆漆·朱内·有印泥
  装饰·朱印 click 时印泥盒亮
  hover·浮·"朱 泥"

[折子叠]·镇纸抬起后才显示
  默认隐藏·镇纸 click 后浮起

[卷轴]·已展开 1 卷 (近期诏书·留念) + 半卷起 1 卷
  装饰·hover 浮·"前诏·X月X事"
  click·展开看·非主操作

[茶杯]·一只
  装饰·hover 浮·朝代茶名 ("龙凤团 / 蒙顶 / 武夷")
  click·喝茶·能量小幅恢复 (装饰交互)

[香炉]·小铜·烟形
  状态·烟形按时辰 (子时纤·午时浓·亥时淡)
  hover·浮·"时 辰·辰 时" + 12 时辰圆盘 (mini)
  对应·rail 右"辰"

[漏壶]·铜壶滴漏 (3 层)
  状态·水位按时辰 + 当日时段
  装饰·桌左角·小

[杯·盘]·装饰
[镇尺·小]

## §6·主题 7 套·**御书房自变·非 hue-rotate**

```
default·    暖深褐底·宋画屏风·明朱漆案·砚黑·朱泥红 (基线)
paper·      净宣阅读·屏风净宣·光照足·朝代衰但 reading-friendly
scroll·     拓本古卷·御书房显古意·屏风换碑拓·桌换青铜古朴
blue·       青花瓷·瓷器陈设·屏风画青花·朝服青
celadon·    汝窑天青·瓷为主调·桌换青玉·屏风山水青绿
vermillion· 朱印宫墙·朝代盛世·朱漆全·烫金满·朱泥多·龙袍重
highcontrast· a11y·CSS only·御书房保持但对比度强化

主题切换 = 御书房布置全换·**非 CSS filter**
工作量·~1 御书房 main view × 7 主题 = 7 张 + 物品 N × 7 次微调
风险·~5-7×资产·但视觉冲击远超 hue-rotate
```

**这是 D1 重于 hue-rotate 的关键**·**主题不是滤镜·是朝代陈设**·

## §7·responsive·桌面优先 + mobile fallback

```
桌面 (>1280)·   御书房 1280×800 · 视域 cover
平板 (768-1280)·御书房 cover·CSS 缩
