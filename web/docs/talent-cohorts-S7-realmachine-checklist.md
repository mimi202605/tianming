## 〇 · 如何开启（flag 默认关·零回归）

引擎全程门控于 `P.conf.talentCohortEnabled`。真机开启两法：

1. **设置面板 toggle**（已加·推荐）：⚙️设置 → **🧪 实验模式**（先点「开启实验模式」选 **LLM 模式**）→ 勾选「**🎓 人才范式渗透（默认关·实验）**」。即时生效、随存档持久。（与 🏛️官制活化、🎭事件系统统一 并列。）
2. **控制台**（dev·等价）：
   ```js
   P.conf.talentCohortEnabled = true; if (typeof saveP === 'function') saveP();
   ```

> 调参集中在各模块顶部 `TUNING`：`tm-talent-cohorts.js`（师资生师比/历练回合/衰减/分档）、`tm-talent-bottlenecks.js`（岗位密度 DENSITY / 动态 room ROOM）、`tm-talent-backlash.js`（阈值/冷却/临界）。手感不对就调这里。

## 一 · 建一座新式学堂（喂引擎）

1. 任选一地 → **兴造** → **自拟营造** 页签。
2. 名目如「格致学堂」「算学馆」「讲武堂」；类属选文化/行政等；规制描述写明「习某某之学、为国育某类人才」。
3. 有 API key + `customBuildAgentEnabled` 时点 **请有司核议** —— 有司应：
   - 勘报含 **〔学统〕** 段（既有范式/旧学存量/本地士绅）。
   - 核议产出 **talentSource**（判这校教什么「学」：新立范式 or 归入已有 / influenceProfile 利于什么 / absorptionKind 需何岗位 / 年毕业数）。
   - 变革阻力走 **globalRule.resistance**（士绅旧党反对·复用国是·风气齿轮）。
4. 准奏开工 → 过工期完工 → 经 S2 桥接把毕业生注入引擎。

## 二 · 推回合·逐点验（开 flag 后）

| # | 验证点 | 期望（硬核可信） | 看哪 |
|---|---|---|---|
| 1 | **渗透滞后** | 学堂完工后渗透**不**立刻跳；要招生→历练数年→才入成熟存量，渗透缓爬。 | 兴造弹窗「人才与风气」面板·渗透% |
| 2 | **师资瓶颈** | 早期 stock≈0 → 师资稀 → 培养质量低（水货）→ 有效人才远小于成熟数。狂建空壳学堂无用。 | 面板·培养质量% / 有效 vs 成熟 |
| 3 | **产业吸纳瓶颈** | 没对应产业（如纯农业区建工技学堂）→ 毕业即失业，失业数涨、渗透不涨。 | 面板·失业数 |
| 4 | **制度空间飞轮** | 旧式当道时新人受压（room 低·有效打折）；渗透起来后 room 升、渗透加速（正反馈）。 | 多回合看渗透加速度 |
| 5 | **临界非线性瓦解** | 渗透越临界（~主流/主导）→ 旧式正统**加速边缘化**（存量加速跌）+「蔚然成风」事件栏。 | 面板·既有正统存量随回合骤跌 |
| 6 | **右·旧势力反扑** | 渗透升快 + 旧式庞大 → **御案时政**出「**旧学之党请罢新学**」（关键决策·3 选项）·AI 据你的选择裁决后果。 | 御案时政/待决时政 |
| 7 | **左·失业动荡** | 建太快太空 → 失业积聚 → 御案时政出「**士子失业·学潮渐起**」。 | 御案时政 |
| 8 | **AI 叙事跟人才态** | 回合推演里 AI 的改革/科技/民意裁决，应**呼应**当前「人才与风气」段（渗透高则改革/实学类更易成）。注意是「成功率倾向」非「必成」。 | 推演叙事/诏令执行结果 |

## 三 · 跨朝代通用性验（铁律③范式自由）

另起一局或另一地，自拟一个**架空学派**（如「玄理之学」「海道经世学」——非「实学/西学」），描述其纲领，看：
- 有司能为它判 influenceProfile（利于什么）、立为新范式。
- 它走**同一引擎**渗透/受阻，与「格致」无二。
- 证引擎**不预设任何具体的「学」**（铁律③）。

## 四 · 剧本 preset（既有正统·可选）

要让某剧本开局就有「旧学正统」庞大存量（新学要从其阴影下爬升），在剧本 JSON 加：
```json
"talentParadigms": [
  { "label": "科举经义", "kind": "established", "stock": 800000 }
]
```
（`tm-patches.js` doActualStart 已支持·flag 开才播种·跨朝代 label 取剧本·官方剧本是否加由你定·加了也只在 flag 开时生效。）
