帮 助 · 入 门
天命者，AI 历史模拟推演也
一如何开始
「开卷」择一剧本，定难度与史笔之格，即入推演。每回合下「诏书」、阅「奏疏」、开「朝议」，再「诏付有司」过回合。
二核心循环
观局势 → 拟诏令 → 问对群臣 → 过回合 → 阅史记。AI 依你之决断推演天下牵动。
三常用快捷
Enter 开卷 · Ctrl+S 存档 · Ctrl+1~9 切栏 · Esc 暂停 · F1 帮助
四需先配 API
于「典章 · API 连接」填入密钥（Anthropic / 中转）。无密钥亦可开局浏览，但推演需 AI。
